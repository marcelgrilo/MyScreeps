Global Objects

Game
Properties
Game.constructionSites object A hash containing all your construction sites with their id as hash keys.
Game.cpu object An object containing information about your CPU usage with the following properties:
Game.creeps object A hash containing all your creeps with creep names as hash keys.
Game.flags object A hash containing all your flags with flag names as hash keys.
Game.gcl object Your Global Control Level, an object with the following properties :
Game.map Map A global object representing world map.
Game.market Market A global object representing the in-game market.
Game.resources object An object with your global resources that are bound to the account, like subscription tokens.
Game.rooms object A hash containing all the rooms available to you with room names as hash keys.
Game.spawns object A hash containing all your spawns with spawn names as hash keys.
Game.structures object A hash containing all your structures with structure id as hash keys.
Game.time number System game tick counter.
Methods
Game.cpu.getUsed() Get amount of CPU time used from the beginning of the current game tick.
Game.getObjectById(id) Get an object with the specified unique ID.
Game.notify(message, [groupInterval]) Send a custom message at your profile email.
Map
Methods
Game.map.describeExits(roomName) List all exits available from the room with the given name.
Game.map.findExit(fromRoom, toRoom, [opts]) Find the exit direction from the given room en route to another room.
Game.map.findRoute(fromRoom, toRoom, [opts]) Find route from the given room to another room.
Game.map.getRoomLinearDistance(roomName1, roomName2, [continuous]) Get the linear distance (in rooms) between two rooms.
Game.map.getTerrainAt(x, y, roomName) Get terrain type at the specified room position.
Game.map.isRoomAvailable(roomName) Check if the room is available to move into.
Market
Properties
Game.market.credits number Your current credits balance.
Game.market.incomingTransactions array An array of the last 100 incoming transactions to your terminals with the following format:
Game.market.outgoingTransactions array An array of the last 100 outgoing transactions from your terminals with the following format:
Game.market.orders object An object with your active and inactive buy/sell orders on the market.
Methods
Game.market.calcTransactionCost(amount, roomName1, roomName2) Estimate the energy transaction cost of StructureTerminal.
Game.market.cancelOrder(orderId) Cancel a previously created order.
Game.market.changeOrderPrice(orderId, newPrice) Change the price of an existing order.
Game.market.createOrder(type, resourceType, price, totalAmount, [roomName]) Create a market order in your terminal.
Game.market.deal(orderId, amount, [yourRoomName]) Execute a trade deal from your Terminal to another player's Terminal using the specified buy/sell order.
Game.market.extendOrder(orderId, addAmount) Add more capacity to an existing order.
Game.market.getAllOrders([filter]) Get other players' orders currently active on the market.
Game.market.getOrderById(id) Retrieve info for specific market order.
Memory
An object which contains user-defined custom memory data. Learn more about memory
PathFinder
Methods
PathFinder.search(origin, goal, [opts]) Find an optimal path between origin and goal.
PathFinder.use(isEnabled) Specify whether to use this new experimental pathfinder in game objects methods.
PathFinder.CostMatrix() Creates a new CostMatrix containing 0's for all positions.
set(x, y, cost) Set the cost of a position in this CostMatrix.
get(x, y) Get the cost of a position in this CostMatrix.
clone() Copy this CostMatrix into a new CostMatrix with the same data.
serialize() Returns a compact representation of this CostMatrix which can be stored via JSON.
PathFinder.CostMatrix.deserialize(val) Static method which deserializes a new CostMatrix using the return value of serialize.
RawMemory
Allows to implement your own memory stringifier instead of built-in serializer based on JSON.stringify. The object is accessible via the global RawMemory variable.
Methods
RawMemory.get() Get a raw string representation of the Memory object.
RawMemory.set(value) Set new memory value.

Room
Properties
controller StructureController The Controller structure of this room, if present, otherwise undefined.
energyAvailable number Total amount of energy available in all spawns and extensions in the room.
energyCapacityAvailable number Total amount of energyCapacity of all spawns and extensions in the room.
memory any A shorthand to Memory.
mode string One of the following constants:
name string The name of the room.
storage StructureStorage The Storage structure of this room, if present, otherwise undefined.
terminal StructureTerminal The Terminal structure of this room, if present, otherwise undefined.
Methods
Room.serializePath(path) Serialize a path array into a short string representation, which is suitable to store in memory.
Room.deserializePath(path) Deserialize a short string path representation into an array form.
createConstructionSite(x, y, structureType) Create new ConstructionSite at the specified location.
createFlag(x, y, [name], [color], [secondaryColor]) Create new Flag at the specified location.
find(type, [opts]) Find all objects of the specified type in the room.
findExitTo(room) Find the exit direction en route to another room.
findPath(fromPos, toPos, [opts]) Find an optimal path inside the room between fromPos and toPos using A* search algorithm.
getPositionAt(x, y) Creates a RoomPosition object at the specified location.
lookAt(x, y) Get the list of objects at the specified room position.
lookAtArea(top, left, bottom, right, [asArray]) Get the list of objects at the specified room area.
lookForAt(type, x, y) Get an object with the given type at the specified room position.
lookForAtArea(type, top, left, bottom, right, [asArray]) Get the list of objects with the given type at the specified room area.
RoomPosition
Properties
roomName string The name of the room.
x number X position in the room.
y number Y position in the room.
Methods
RoomPosition(x, y, roomName) You can create new RoomPosition object using its constructor.
createConstructionSite(structureType) Create new ConstructionSite at the specified location.
createFlag([name], [color], [secondaryColor]) Create new Flag at the specified location.
findClosestByPath(type, [opts]) Find an object with the shortest path from the given position.
findClosestByRange(type, [opts]) Find an object with the shortest linear distance from the given position.
findInRange(type, range, [opts]) Find all objects in the specified linear range.
findPathTo(x, y, [opts]) Find an optimal path to the specified position using A* search algorithm.
getDirectionTo(x,y) Get linear direction to the specified position.
getRangeTo(x,y) Get linear range to the specified position.
inRangeTo(x, y, range) Check whether this position is in the given range of another position.
isEqualTo(x,y) Check whether this position is the same as the specified position.
isNearTo(x,y) Check whether this position is on the adjacent square to the specified position.
look() Get the list of objects at the specified room position.
lookFor(type) Get an object with the given type at the specified room position.

Creep
Inherited from: RoomObject
Properties
body array An array describing the creep’s body.
carry object An object with the creep's cargo contents.
carryCapacity number The total amount of resources the creep can carry.
fatigue number The movement fatigue indicator.
hits number The current amount of hit points of the creep.
hitsMax number The maximum amount of hit points of the creep.
id string A unique object identificator.
memory any A shorthand to Memory.
my boolean Whether it is your creep or foe.
name string Creep’s name.
owner object An object with the creep’s owner info containing the following properties:
saying string The text message that the creep was saying at the last tick.
spawning boolean Whether this creep is still being spawned.
ticksToLive number The remaining amount of game ticks after which the creep will die.
Methods
attack(target) Attack another creep or structure in a short-ranged attack.
attackController(target) Decreases the controller's downgrade or reservation timer for 1 tick per every 5 CLAIM body parts (so the creep must have at least 5xCLAIM).
build(target) Build a structure at the target construction site using carried energy.
cancelOrder(methodName) Cancel the order given during the current game tick.
claimController(target) Claims a neutral controller under your control.
dismantle(target) Dismantles any (even hostile) structure returning 50% of the energy spent on its repair.
drop(resourceType, [amount]) Drop this resource on the ground.
generateSafeMode(controller) Add one more available safe mode activation to a room controller.
getActiveBodyparts(type) Get the quantity of live body parts of the given type.
harvest(target) Harvest energy from the source or minerals from the mineral deposit.
heal(target) Heal self or another creep.
move(direction) Move the creep one square in the specified direction.
moveByPath(path) Move the creep using the specified predefined path.
moveTo(x, y, [opts]) Find the optimal path to the target within the same room and move to it.
notifyWhenAttacked(enabled) Toggle auto notification when the creep is under attack.
pickup(target) Pick up an item (a dropped piece of energy).
rangedAttack(target) A ranged attack against another creep or structure.
rangedHeal(target) Heal another creep at a distance.
rangedMassAttack() A ranged attack against all hostile creeps or structures within 3 squares range.
repair(target) Repair a damaged structure using carried energy.
reserveController(target) Temporarily block a neutral controller from claiming by other players and restore energy sources to their full capacity.
say(message, [public]) Display a visual speech balloon above the creep with the specified message.
signController(target, text) Sign a controller with a random text visible to all players.
suicide() Kill the creep immediately.
transfer(target, resourceType, [amount]) Transfer resource from the creep to another object.
upgradeController(target) Upgrade your controller to the next level using carried energy.
withdraw(target, resourceType, [amount]) Withdraw resources from a structure.

Inherited from: RoomObject, Structure, OwnedStructure
